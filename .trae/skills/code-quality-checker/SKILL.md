# 代码质量检查器

## 功能

- **代码简洁度分析**：评估代码的可读性、复杂度和冗余度
- **运行效果检查**：验证代码的执行结果是否符合预期
- **性能评估**：分析代码运行时间和资源使用情况
- **最佳实践检查**：识别不符合编码规范的问题

## 使用场景

- 当用户要求检查代码质量时
- 当用户需要验证代码运行效果时
- 当用户希望优化代码性能时
- 当用户请求代码审查时

## 检查内容

### 代码简洁度
- 代码行数和复杂度
- 函数和变量命名规范
- 重复代码检测
- 注释质量和完整性
- 代码结构和组织

### 运行效果
- 执行结果正确性
- 错误处理和边界情况
- 性能瓶颈识别
- 内存使用情况
- 执行时间分析

## 输出格式

提供详细的检查报告，包括：
- 问题列表和严重程度
- 改进建议
- 性能分析结果
- 代码质量评分

## 支持的语言

- JavaScript/TypeScript
- Python
- Java
- C/C++
- Go
- Rust

## 使用示例

当用户提交代码并请求检查时，该技能会自动分析代码质量并提供详细报告。
