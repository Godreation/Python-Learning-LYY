# 几何物理沙盒模拟器

一个基于Python和Pygame的物理沙盒模拟器，展示了粒子系统在中心引力、鼠标斥力和边界力作用下的运动效果。

## 功能特性

- **中心引力**：粒子会被中心吸引子吸引
- **鼠标斥力**：鼠标移动时会对周围粒子产生斥力
- **边界约束**：粒子无法超出设定的边界范围
- **粒子碰撞**：粒子间会发生弹性碰撞
- **实时状态显示**：显示帧率、粒子数和系统能量
- **交互控制**：支持暂停/继续、重置、调整引力等操作

## 控制说明

- **空格**：暂停/继续模拟
- **R**：重置粒子系统
- **A**：增加引力强度
- **Z**：减少引力强度
- **Q**：退出程序

## 依赖库

- Python 3.7+
- Pygame 2.0+

## 安装与运行

1. 安装依赖库：
   ```bash
   pip install pygame
   ```

2. 运行程序：
   ```bash
   python geometric_sandbox.py
   ```

## 技术实现

- 使用欧拉积分法更新粒子状态
- 采用双缓冲技术提高渲染性能
- 实现了简单的碰撞检测与响应
- 支持60FPS的稳定帧率
