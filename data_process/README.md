# 实验数据处理系统

一个基于Python的简洁GUI实验数据处理系统，专门用于处理少量经过格式化与去重的实验数据，满足一般论文的数据处理需求。

## 功能特点

### 📊 数据导入与导出
- **导入格式**: 支持 `.xlsx` 和 `.xls` 格式的Excel文件
- **导出格式**: 支持将处理后的数据导出为 `.xlsx` 格式
- **数据预览**: 以表格形式实时显示导入的数据

### 📈 数据分析功能
- **统计描述**: 提供数据的统计描述信息（均值、标准差、最小值、最大值、四分位数等）
- **数据预览**: 实时显示数据结构和内容

### 🎨 数据可视化
- **图表类型**: 支持折线图、散点图、柱状图、直方图
- **坐标轴选择**: 可自定义选择X轴和Y轴数据列
- **图片保存**: 支持将图表保存为PNG、JPG、SVG格式
- **高分辨率**: 图片保存支持300DPI高分辨率输出

### 🖥️ 用户界面
- **简洁易用**: 直观的GUI界面，操作简单
- **实时反馈**: 数据大小和状态实时显示
- **响应式设计**: 支持窗口大小调整

## 系统要求

### Python版本
- Python 3.7+

### 依赖包及版本要求

| 包名 | 版本要求 | 功能说明 |
|------|----------|----------|
| pandas | >= 2.0.0 | 数据处理和分析核心库 |
| openpyxl | >= 3.0.0 | Excel文件读写支持 |
| matplotlib | >= 3.5.0 | 数据可视化图表绘制 |
| numpy | >= 1.20.0 | 数值计算支持 |
| tkinter | Python内置 | GUI界面框架 |

### 推荐安装命令
```bash
pip install pandas>=2.0.0 openpyxl>=3.0.0 matplotlib>=3.5.0 numpy>=1.20.0
```

## 安装与使用

### 1. 安装依赖包
确保已安装Python 3.7+，然后安装所需依赖：

```bash
pip install pandas openpyxl matplotlib numpy
```

### 2. 运行程序
```bash
python data_processor.py
```

### 3. 使用步骤
1. **导入数据**: 点击"导入Excel数据"按钮选择Excel文件
2. **查看数据**: 在左侧表格中预览导入的数据
3. **数据分析**: 点击"统计描述"查看数据统计信息
4. **数据可视化**: 
   - 选择图表类型（折线图/散点图/柱状图/直方图）
   - 选择X轴和Y轴数据列
   - 点击"生成图表"查看可视化结果
5. **导出结果**: 
   - 点击"导出Excel数据"保存处理后的数据
   - 点击"保存图片"将图表保存为图片文件

## 适用场景

- 🔬 **实验数据处理**: 适合处理实验室产生的少量实验数据
- 📚 **论文写作**: 满足一般学术论文的数据处理和图表需求
- 📊 **数据分析教学**: 适合数据分析初学者学习和使用
- 🔍 **数据探索**: 快速查看和分析格式化数据

## 注意事项

1. **数据量限制**: 建议处理的数据量不超过1000行，以保证最佳性能
2. **数据格式**: 建议数据已经过格式化和去重处理
3. **文件格式**: 支持标准的Excel格式文件
4. **图表保存**: 图片保存支持多种格式，推荐使用PNG格式
