# 个人财务管理系统

一个功能完善的个人财务管理应用，帮助用户追踪收入支出，分析消费习惯，制定预算计划。

## 功能特性

### 核心功能
- ✅ **交易记录管理** - 添加、编辑、删除收入支出记录
- ✅ **分类系统** - 支持多种交易类别和标签
- ✅ **自动分类** - 基于关键词匹配自动分类交易
- ✅ **数据分析** - 按月/年统计收支情况，生成图表
- ✅ **预算管理** - 创建预算计划，实时追踪进度
- ✅ **数据持久化** - 使用SQLite数据库存储数据
- ✅ **导入导出** - 支持CSV/Excel格式数据导入导出
- 🔄 **报表生成** - 生成月度/年度财务报告（开发中）

### 技术特点
- **现代化GUI** - 基于Tkinter的直观用户界面
- **数据安全** - 自动备份和恢复功能
- **智能分析** - 异常消费检测和财务健康评分
- **可扩展性** - 模块化设计，易于功能扩展

## 安装指南

### 系统要求
- Python 3.7+
- Windows/macOS/Linux

### 安装步骤

1. **克隆或下载项目**
   ```bash
   cd d:\Code_Learning\finance_manager
   ```

2. **安装依赖包**
   ```bash
   pip install pandas matplotlib openpyxl
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 快速开始（一键运行）
项目包含自动依赖检查功能，首次运行时会自动检测并提示安装缺失的依赖包。

## 使用说明

### 首次使用
1. 运行程序后，系统会自动创建数据库并初始化默认分类
2. 开始添加您的第一笔交易记录
3. 设置预算计划来追踪支出

### 主要功能操作

#### 添加交易
1. 点击"添加交易"菜单
2. 填写金额、类型（收入/支出）、分类、描述
3. 系统会自动根据描述关键词进行智能分类

#### 查看仪表盘
- 实时显示本月收支情况
- 最近交易记录
- 预算提醒和财务健康评分

#### 预算管理
1. 创建月度预算计划
2. 系统自动追踪预算使用情况
3. 超过阈值时发出提醒

#### 数据分析
- 查看收支趋势图表
- 分析消费习惯
- 生成财务健康报告

## 数据格式说明

### 交易记录字段
- **日期**：YYYY-MM-DD格式
- **类型**：收入(income) / 支出(expense)
- **金额**：正数，支持小数点后两位
- **分类**：预定义的收入/支出类别
- **描述**：交易描述，用于自动分类
- **标签**：可选的标签标记

### 支持的文件格式
- **导入**：CSV, Excel (.xlsx, .xls)
- **导出**：CSV, Excel, JSON

## 故障排除

### 常见问题

**Q: 程序启动时报错 "ModuleNotFoundError"**
A: 请运行 `pip install pandas matplotlib openpyxl` 安装依赖包

**Q: 数据库文件损坏怎么办？**
A: 系统会自动创建备份，可以尝试从备份恢复

**Q: 导入CSV文件失败**
A: 请检查CSV文件格式，确保包含必要的列：date, type, amount, description

### 获取帮助
如遇到问题，请检查：
1. Python版本是否为3.7+
2. 所有依赖包是否已正确安装
3. 文件路径是否包含中文或特殊字符
